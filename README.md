# Powerful Progress — Landing

Single-page waitlist site. Plain HTML/CSS/JS. No build step.

## Local preview

Open `index.html` directly, or serve it:

```bash
python3 -m http.server 8080
# then visit http://localhost:8080
```

## Deploy

Vercel will auto-detect this as a static site. `vercel.json` is included for headers.

## Files

- `index.html` — entire site (HTML, CSS, JS inline)
- `vercel.json` — Vercel config
